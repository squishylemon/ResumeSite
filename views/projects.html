Game Development

Here are some of the game development projects that I have worked on:

  • Grade 6 (2019) Unity Game - Ball Control

    In this passion project, I developed a simple game in Unity where the player controls a ball and can spawn a set number of AI-controlled balls to chase them. The game is set on a map with a hole in the middle, and the player must get all the AI balls to fall into the hole to earn points. The game never ends, and the player's highest score is tracked until they exit the game.

  • Grade 8 (2021) Unity Game - Wuhu Island Remastered

    For this home project, I set out to remaster the Wuhu Island map from Wii Sports Resort in Unity using the High Definition Render Pipeline (HDRP) and testing out new features such as terrain building and creating realistic water. The project is still ongoing, but you can see some of the progress on my website.
